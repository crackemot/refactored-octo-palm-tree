# Детектор синтетической речи (VC/STS)

## Описание
Проект представляет собой программную реализацию детектора синтетической и преобразованной речи, создаваемой с использованием технологий Voice Conversion и Speech-to-Speech Synthesis (VC/STS).

В основе решения лежит интерпретируемый эвристический подход с автоматической калибровкой параметров. Проект не использует нейросетевые модели и не требует больших обучающих выборок.

---

## Структура проекта
- `calibrator.py` — модуль автоматической калибровки пороговых значений и весов эвристик на основе калибровочной выборки реальных и синтетических аудиозаписей.
- `detector.py` — модуль анализа аудиозаписей и формирования итоговой оценки и классификационного вердикта.
- `fake/` — каталог с заведомо синтетическими аудиофайлами для калибровки.
- `real/` — каталог с заведомо реальными аудиофайлами для калибровки.

---

## Принцип работы
Работа системы разделена на два этапа:

1. **Калибровка**  
   Вычисление статистических характеристик акустических признаков и автоматический подбор параметров детектора на основе калибровочной выборки.

2. **Детекция**  
   Анализ аудиозаписи, эвристическая оценка и принятие решения о принадлежности записи к реальной или синтетической речи.

---

## Использование

Запуск калибровки:
```bash
python calibrator.py
```
Запуск детектора:

```bash
python detector.py [-h] [-c CONFIG] [-v] [-q] FILE [FILE ...]
```

Результаты работы детектора выводятся в консоль в виде численной оценки и классификационного вердикта.

## Установка зависимостей

```bash
pip install -r requirements.txt
```
